The WebSocket protocol defines a [url=ws://]ws:[/url] and [url=wss://]wss:[/url] prefix to indicate a WebSocket and a WebSocket Secure connection, respectively. Both schemes use an HTTP upgrade mechanism to upgrade to the WebSocket protocol. Some proxy servers are harmless and work fine with WebSockets; others will prevent WebSockets from working correctly, causing the connection to fail.
