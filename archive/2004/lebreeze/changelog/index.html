Changelog

This page lists all changes made to the unofficial version of LeBreeze.

  • Bold entries mark major changes.
  • Red entries mark changes that only affect the Firefox theme.
  • Blue entries mark changes that only affect the Thunderbird theme.

18 June 2004

  • Full Firefox 0.9 compatibility

20 May 2004 - Version 1.1

  • New UI: Extension and Theme Managers
  • Firefox 0.9 Theme API compatibility
  • Revised UI: Options
  • Revised UI: Customize Toolbar
  • Misc: Optimized code. Reduced filesize.

06 May 2004

  • Revised UI: Color Picker
  • Bugfix: Disabled buttons respond when clicked
  • Misc: Made URL bar and search bar automatically narrow to match height of small icons

05 May 2004

  • Misc: Disabled states for toolbar buttons are now being rendered with -moz-opacity

03 May 2004

  • Revised UI: Customize Toolbar
  • New theme option: Compact menus
  • Misc: Widened the search bar

02 May 2004

  • New extension support: Basics
  • New theme option: Display tab icons
  • New theme option: “Breeze” bookmark and folder icons
  • New theme option: “Play” toolbar icons [Removed 20/May/04]
  • Miscellaneous cosmetic improvements
  • Miscellaneous bugfixes
  • Misc: Removed source PSDs and unused graphics from jarfile. Reorganized directory structure. Filesize reduced significantly.
